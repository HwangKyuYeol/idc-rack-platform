# IDC 전력 계산기

IDC Rack Power Platform은 서버, 네트워크 장비, 보안장비, 스토리지의 랙 전력과 PDU 부하를 계산하는 웹 기반 도구입니다.

## 주요 기능

- IDC 전력 계산
- 서버 전력 계산
- 랙 전력 계산
- PDU A/B 전원 계산
- 데이터센터 랙 배치도 작성
- 서버, 네트워크, 보안장비, 스토리지 전력 산정
- PDF, PNG, CSV 보고서 출력

## 대표 검색 키워드

IDC 전력 계산, 서버 전력 계산, 랙 전력 계산, PDU 전력 계산, 데이터센터 전력 계산, 서버 랙 배치도
